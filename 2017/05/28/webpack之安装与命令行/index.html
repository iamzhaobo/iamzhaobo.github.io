webpack之安装与命令行

发表于 2017-05-28   |   659 字   |   3 min | 分类于 框架 |

webpack简介

webpack是一个代码打包器，依附于NPM平台。在前端领域，所谓的代码打包，即是将所有模块化后的代码打包到一个JS文件中，HTML中只需引入这个打包后的文件即可，实际在浏览器中运行的也是这一个文件。

除了对JS文件进行打包之外，还可以对其他的非JS文件进行打包，如打包后的CSS引入JS文件中。

当然除了打包这一功能之外，webpack还有其他的功能。在模块化上，它支持CommonJS、AMD、CMD、ES6多种模块化方式，并且可以在一个功能中混用。依附于webpack同样有相当多的插件，用来实现网页的热部署，利用babel来讲ES6转ES5等。

接下来，将介绍webpack的简单使用与相关命令。

webpack代码的编写

新建工程webpack-test

利用npm初始化整个工程

在CMD中

npm init // 命令行中与填写的字段全部Enter跳过

之后，即可在工程根目录下生成package.json文件

安装webpack

在CMD中

npm install webpack --save-dev

安装完成之后，与webpack相关的依赖将会存放在工程根目录下的node_modules文件夹中

安装完成

至此，在当前工程下的webpack环境就安装完成。

webpack命令

webpack打包

  1. 在根目录下新建文件hello.js

    import './world.js'

function hello(str) {
    alert(str);
}

hello('Test Webpack')

  2. 在根目录下新建文件world.js

    function world() {
    return {

    }
}

  3. 在CMD中执行命令

    webpack hello.js hello.bundle.js

    执行上述操作后，即可将hello.js和与之关联的文件进行打包，打包文件为hello.bundle.js，这个bundle文件存放在工程的根目录下；

  4. 通过HTML文件对之进行引用，新建index.html,对hello.bundle.js进行引用

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Demo</title>
</head>
<body>
    <h1>webpack</h1>
    <script type="text/javascript" src="hello.bundle.js"></script>
</body>
</html>

打开网页后，在浏览器中就会有一个“Test Webpack”的alert。

webpack打包CSS

  1. 新建样式文件style.css

    body {
    background: red;
}

  2. 引入需要的依赖

    // 在CMD中
    npm install style-loader css-loader

  3. 引入CSS文件

    1. 在hello.js中引入

      import 'style-loader!css-loader!./style.css';

    2. CMD中加载

      1. hello.js

        import './style.css';

      2. CMD命令中

        webpack hello.js hello.bundle.js --module-bind "css=style-loader!css-loader"

热部署

webpack hello.js hello.bundle.js --watch

观察打包过程

webpack hello.js hello.bundle.js --progess

列出所有打包的模块(包括处理方式)

webpack hello.js hello.bundle.js --display-modules

加载模块的原因

webpack hello.js hello.bundle.js  --display-reasons
